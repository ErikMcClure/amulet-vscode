# Amulet VS Code

**Amulet support for [Visual Studio Code](https://code.visualstudio.com/)**

## Requirements
 - The extension requires Amulet to be already be installed.

## Features
 - Syntax highlighting
 - Error highlighting
 - Automatic hole-filling
 - Show symbols in file
